# 权限介绍

* `rwx`权限详解

  | 权限 | 作用到文件                                           | 作用到目录                                 |
  | :--: | ---------------------------------------------------- | ------------------------------------------ |
  | `r`  | 表示可以读取                                         | 可以读取，可以查看(用ls指令)目录包含的文件 |
  | `w`  | 表示可以修改不代表可以删除(可以修改是可以删除的前提) | 可以修改，目录内可以创建+删除+重命名目录   |
  | `x`  | 可以执行                                             | 可以进入目录                               |

  

* `rwx`可用数字表示
  * `r`：4
  * `w`：2
  * `x`：1
  * `无权限`：0
  * `rwx`=4+2+1=7
