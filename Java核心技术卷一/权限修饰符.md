# 权限修饰符

|    修饰符     |  类  |  包  | 子类 | 其他包 |
| :-----------: | :--: | :--: | :--: | :----: |
|  **public**   |  √   |  √   |  √   |   √    |
| **protected** |  √   |  √   |  √   |   ×    |
|  **default**  |  √   |  √   |  ×   |   ×    |
|  **private**  |  √   |  ×   |  ×   |   ×    |

* **protected，default，private**在其他包中都是不能访问的
  * protected可以将属性继承给子类 不管在不在同一个包中
  * default在同一个包内可以将属性继承给子类，在不同包中则不能继承给子类
  * private不能将属性继承给子类
